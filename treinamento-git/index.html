4.2- Qual a diferença entre git e github?

O GIT é uma espécie de software que possibilita realizar o controle de versão de algo que esteja sendo monitorado por ele.
Por meio dele é possível realizar o controle de cada detalhe do histório das coisas que sofreram algum tipo de ajuste. Caso seja necessário é possível visualizar as pessoas que realizaram as alterações ou executar uma espécie de restauração, ou seja voltar alguma versão anterior

GIThub é uma espécie de plataforma que possibilita de forma visual fazer a utilização do GIT, por meio dele é possível realizar o compartilhamento e controle de tudo o que foi gerenciado pelo Git de forma visual.
